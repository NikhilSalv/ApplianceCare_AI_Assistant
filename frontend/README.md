# Frontend - Next.js Application

This is the Next.js frontend application for the ApplianceCare AI Assistant.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`

## Build for Production

```bash
npm run build
npm start
```

## Features

- Search interface for querying appliance repair information
- Displays AI-generated answers with confidence scores
- Responsive design for mobile and desktop
- Built with Next.js 14 and JavaScript (no TypeScript)

## Configuration

Make sure the backend API is running on `http://localhost:8000` or update the API URL in `app/page.jsx` if using a different port.
